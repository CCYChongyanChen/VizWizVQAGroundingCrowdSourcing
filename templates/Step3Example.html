If the question asks about the object: draw the outside boundary of the object (Example 1). Usually, questions look like “what is this?”.

If the question asks about the specific details (a property) of the object: draw the boundary of the related area. (Example 2 & Example 3 & Example 4).


Tips:

(1) You can first identify if the object is mentioned in the question. If the object is mentioned in the question, usually it asks about the property of the object (Example 2 & Example 3 & Example 4).

(2) If the question asks about specific details (a property) of a visual entity (e.g., color of clothes), please comprehensively annotate all regions that would lead to the answer rather than a minimum visible region (Example 2 & Example 3 & Example 4).

(3) Question may use other object(s) to refer to the target object (e.g., "What is the color of the bag next to the man's hand on the table?"). Please only draw the target object and nothing else.
